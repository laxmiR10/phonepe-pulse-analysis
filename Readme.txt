# PhonePe Pulse Data Analysis

## Objective
Analyze PhonePe transaction, user, and insurance data to derive business insights.

## Tools Used
Python, MySQL, Streamlit, Pandas, Matplotlib

## Data Source
PhonePe Pulse GitHub Repository

## Methodology
- Data Extraction (JSON)
- Data Transformation
- SQL Storage
- Business Analysis
- Visualization
- Dashboard Creation

## Key Insights
- State-wise transaction dominance
- User engagement trends
- District-level performance

## Conclusion
The project demonstrates practical data engineering and analytics skills with real-world data.
